Informática:
1- ¿Cuál es la función principal de la memoria ram? a) Guardar archivos permanentes. b) Ejecutar operaciones matemáticas. c) Almacenar datos/programas temporalmente mientras se usan. d) Controlar dispositivos externos.
Respuesta correcta: C. La memoria RAM guarda información de manera temporal para que el procesador pueda acceder rápidamente mientras la computadora está funcionando.
2- ¿Cuál es la función principal del procesador (CPU) en una computadora? a) Almacenar archivos permanentemente b) Ejecutar instrucciones y procesar datos c) Mostrar imágenes en pantalla d) Conectar dispositivos externos
Respuesta correcta: B. El procesador es el componente encargado de realizar cálculos y ejecutar las instrucciones necesarias para el funcionamiento de la computadora
3- ¿Para qué se utilizan los símbolos y flechas en un diagrama de flujo? a) Para decorar el algoritmo b) Para representar el paso a paso del algoritmo c) Para guardar información en la memoria d) Para reiniciar el algoritmo
Respuesta correcta: B. Para hacer un diagrama de flujo se usan símbolos y flechas mostrando el paso a paso de un algoritmo.
4- ¿Qué símbolo se usa para finalizar una instrucción en lenguaje C? a) . b) : c) . d) ;
Respuesta correcta: D. En C, cada instrucción debe terminar con punto y coma para indicar el final de la línea de código
5- ¿Qué estructura se utiliza para repetir instrucciones varias veces en un programa? a) Variable b) Clase c) Librería d) Bucle
Respuesta correcta: D. Un bucle permite ejecutar varias veces un bloque de instrucciones mientras se cumpla una condición determinada.

Respuesta correcta: C) OR, ya que solamente da "0" cuando ambas entradas son "0".
2. ¿Qué hace un mapa de Karnaugh?
A) Convierte números binarios en decimales B) Simplificar funciones lógicas C) Suma señales eléctricas D) Amplifica voltaje
Respuesta correcta: B) Simplificar funciones lógicas. Permite reducir la cantidad de compuertas necesarias en un circuito, agrupando combinaciones de valores iguales. Gracias a esto, los circuitos son más simples, económicos y eficientes.
3. ¿Qué estudia el álgebra de Boole?
A) Circuitos eléctricos de potencia B) Operaciones matemáticas con números complejos C) El comportamiento lógico de variables binarias D) El movimiento de electrones
Respuesta correcta: C) El comportamiento lógico de variables binarias. Se utiliza para analizar y diseñar circuitos digitales mediante operaciones lógicas como AND, OR y NOT. Es la base matemática de la electrónica digital y de las computadoras.
